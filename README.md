# GadgetHeaven✨ - E-commerce Platform

An interactive React-based e-commerce site designed for gadget enthusiasts. Explore gadgets, manage your cart and wishlist, and navigate seamlessly with a polished interface. GadgetHeaven provides a smooth shopping experience with real-time notifications, budget control, and dynamic layouts.

---

## 🌟 Overview

*GadgetHeaven✨* is a single-page app offering a variety of gadgets with unique functionalities:
- Users can browse, add to cart or wishlist, view product stats, and receive feedback through notifications.
- Manage cart and wishlist with price limits and sorting features.
- Persistent data ensures selections remain intact across sessions.

---

## ✨ Key Features

1. **Wishlist & Cart**: Add gadgets to wishlist or cart, with real-time notifications and automatic duplicate prevention.
2. **Cart with Limits**: Total cart cost capped at $1000; error toast appears if exceeded.
3. **Notifications**: Feedback when adding items to the cart or wishlist.
4. **Dynamic Navbar**: Active routes and different page backgrounds for enhanced UX.
5. **Statistics & Sorting**: Dashboard includes a price chart and price sorting for the cart.

---

## ⚙️ Technologies Used

- **React** for UI and state management.
- **Context API** and **LocalStorage** for data persistence.
- **React Toastify** for real-time notifications.
- **Tailwind CSS** for a responsive, streamlined layout.
- **REcharts** for product pricing graphs.

---

## 📊 Data Handling

Cart and wishlist data are managed with **Context API** and **LocalStorage** for persistence.
